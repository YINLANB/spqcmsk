# 模型下载说明

## 问题

自动下载因网络代理问题失败。请手动下载模型。

## 下载地址

### ProPainter (推荐)
- **Google Drive**: https://drive.google.com/file/d/1OwJBLUjh4wEUv4W9GKqbYMa8dPHqZMw1/view
- **百度网盘**: https://pan.baidu.com/s/1MxLQf8rNkHhRzGNlLZfVwA?pwd=1234
- **GitHub**: https://github.com/sczhou/ProPainter/releases

### GFPGAN (人脸修复，可选)
- **GitHub**: https://github.com/TencentARC/GFPGAN/releases

## 安装步骤

### 方法1: 手动下载

1. 下载模型文件 `propainter.pth`
2. 放到以下目录：
   ```
   video_demosiac/checkpoints/propainter.pth
   ```

### 方法2: 使用 git clone

```bash
cd video_demosiac

# 克隆 ProPainter 仓库
git clone https://github.com/sczhou/ProPainter.git

# 复制模型文件（如果仓库包含预训练模型）
cp ProPainter/experiments/pretrained_models/*.pth checkpoints/
```

### 方法3: 使用 huggingface-cli

```bash
pip install huggingface_hub

# 下载 ProPainter 模型
huggingface-cli download sczhou/Propainter --local-dir ./checkpoints
```

## 验证安装

模型下载后，运行以下命令验证：

```bash
cd video_demosiac
python -c "import os; print('模型文件:', os.path.exists('checkpoints/propainter.pth'))"
```

应该输出: `模型文件: True`

## 使用模型

模型下载后，正常运行即可自动使用：

```bash
python main.py --input video.mp4 --output output.mp4
```

## 常见问题

### Q: 下载速度慢？
A: 使用百度网盘或 Google Drive 下载，速度更快。

### Q: 模型文件多大？
A: ProPainter 约 300MB，GFPGAN 约 350MB。

### Q: 没有 GPU 能用吗？
A: 可以使用 CPU 处理，但速度较慢：
```bash
python main.py --input video.mp4 --output output.mp4 --device cpu
```
